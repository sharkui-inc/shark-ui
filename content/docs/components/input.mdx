## API Reference

### Input

Text input. Accepts all native input attributes.

| Prop | Type | Default |
| --- | --- | --- |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` |
| `type` | `string` | `text` |
| `className` | `string` | `-` |

---

For a complete list of props, see the [Ark UI documentation](https://ark-ui.com/docs/components/field#api-reference).
