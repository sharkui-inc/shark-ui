
## API Reference

| Prop | Type | Default | 
| --- | --- | --- | 
| `variant` | `"default" \| "outline" \| "secondary" \| "ghost" \| "link" \| "destructive"` | `"default"` | 
| `size` | `"xs" \| "sm" \| "md" \| "lg" \| "xl" \| "icon-xs" \| "icon-sm" \| "icon-md" \| "icon-lg" \| "icon-xl"` | `"md"` |
| `clickEffect` | `boolean` | `true` | 
| `isLoading` | `boolean` | `false` |
